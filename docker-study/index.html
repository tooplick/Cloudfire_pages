Docker 命令速查表

Docker 是常用的容器化工具，适合应用打包、环境隔离、服务部署和本地开发测试等场景。

这篇文章主要整理常见 Docker 命令，方便日常使用时快速查阅。

镜像命令

场景 命令
查看本地镜像 docker images
搜索镜像 docker search nginx
拉取镜像 docker pull nginx
删除镜像 docker rmi nginx
构建镜像 docker build -t myapp:latest .
给镜像打标签 docker tag myapp:latest myapp:v1

容器命令

场景 命令
查看运行中容器 docker ps
查看全部容器 docker ps -a
创建并启动容器 docker run -d --name mynginx -p 80:80 nginx
启动容器 docker start 容器名
停止容器 docker stop 容器名
重启容器 docker restart 容器名
删除容器 docker rm 容器名
强制删除运行中容器 docker rm -f 容器名
查看容器详情 docker inspect 容器名

日志与进入容器

场景 命令
查看日志 docker logs 容器名
实时查看日志 docker logs -f 容器名
进入容器 docker exec -it 容器名 /bin/bash
进入 sh 环境 docker exec -it 容器名 /bin/sh
查看容器进程 docker top 容器名

数据卷与文件拷贝

场景 命令
挂载目录启动 docker run -d -v /host:/data 镜像名
查看数据卷 docker volume ls
创建数据卷 docker volume create mydata
删除数据卷 docker volume rm mydata
主机拷贝到容器 docker cp ./test.txt 容器名:/tmp/
容器拷贝到主机 docker cp 容器名:/tmp/test.txt ./

网络命令

场景 命令
查看网络 docker network ls
创建网络 docker network create mynet
删除网络 docker network rm mynet
容器连接网络 docker network connect mynet 容器名
查看网络详情 docker network inspect mynet

系统清理

场景 命令
查看 Docker 磁盘占用 docker system df
删除已停止容器 docker container prune
删除悬空镜像 docker image prune
删除未使用网络 docker network prune
删除未使用数据卷 docker volume prune
一键清理未使用资源 docker system prune
一键清理全部未使用资源 docker system prune -a

危险命令大全

命令 危险点 常见用途
docker rm -f 容器名 强制删除运行中的容器 清理卡住的容器
docker rmi 镜像名 删除镜像后可能无法快速重启环境 清理无用镜像
docker system prune 删除未使用容器、网络、缓存 清理磁盘空间
docker system prune -a 连未使用镜像也删除 彻底清理环境
docker volume prune 删除未使用数据卷，数据可能无法恢复 清理旧卷
docker network prune 删除未使用网络 清理网络资源

Tips:

看到 rm、rmi、prune 这几个关键词时，执行前先确认是否还有数据要保留。
