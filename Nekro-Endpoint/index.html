Nekro Endpoint

在边缘构建你的 API 端点，支持静态内容、代理转发、动态脚本执行。

Github 地址：https://github.com/NekroAI/nekro-endpoint

项目介绍

Nekro Endpoint 是一个基于 Cloudflare Workers 的轻量级端点平台，支持三种端点类型转发：

  • Static：静态内容
  • Proxy：代理转发
  • Script：动态脚本执行

依托 Cloudflare 全球 300+ 节点网络，实现端点的 全球加速 与 低延迟响应。
