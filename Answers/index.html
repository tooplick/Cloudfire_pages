1. 闭包 (Closure)

闭包是指一个函数能够记住并访问其词法作用域，即使该函数在其作用域之外执行。它是由函数以及声明该函数的词法环境组合而成。

常见应用场景：
创建“私有变量”，即通过函数内容的局部变量和返回的内部函数，使得外部无法直接访问该变量，但可以通过内部函数进行操作，从而实现了数据的封装和隐蔽。

2. 块级元素与内联元素

  • <div> 是一个块级元素，通常用于组合其他HTML元素，独占一行，常用于页面布局。
  • <span> 是一个内联元素，用于在文本行内对一部分内容进行分组或样式设置，不会换行。

示例： 使用 <div> 来包裹一个导航栏，用 <span> 来高亮一段文字中的某个关键字。

3. CSS 盒模型 (Box Model)

CSS盒模型是一个用于设计和布局的基本概念，它将每个HTML元素视为一个规矩的盒子。这个盒子由内到外包含四个部分：

  1. 内容区域 (Content)
  2. 内边距 (Padding)
  3. 边框 (Border)
  4. 外边距 (Margin)

4. BOM 与 DOM 的区别

  • BOM (Browser Object Model)：浏览器对象模型，操作浏览器相关功能，比如 window.alert() 弹框。
  • DOM (Document Object Model)：文档对象模型，操作HTML页面元素，比如 document.getElementById() 获取元素。

核心区别： 操作对象不同，一个是浏览器，一个是页面文档。

5. DOCTYPE 声明

<!DOCTYPE html> 声明位于HTML文档的最顶部，它的作用是告知浏览器当前文档使用的是 HTML5 标准。这能确保浏览器以标准模式来渲染页面，避免因旧版本HTML导致的怪异模式，从而保证页面在不同浏览器中显示一致。

6. 浮动 (Float)

浮动的主要用途是实现文字环绕图片的效果，或者让多个块级元素水平排列。

  • 副作用： 导致父元素高度塌陷（父元素无法包裹浮动子元素）。
  • 清除浮动： 方法之一是为父元素添加一个伪元素，并设置 clear:both。
